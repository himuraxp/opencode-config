# Exemple Monorepo

Exemple de structure projet pour un monorepo (Nx, Turborepo, pnpm workspaces) avec `AGENTS.md` et documentation IA.

Généré via :

```bash
~/.config/opencode-config/scripts/init-project.sh
```

## Structure attendue

```txt
monorepo/
  AGENTS.md
  .opencode/
    agents/
      project.md
      security.md
  docs/
    ai/
      PLAN.md
      STATUS.md
      DECISIONS.md
      CHANGELOG.md
  packages/
    ...
```
